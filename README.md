Invisible Internet
==================

You'll need the following installed - 

1) Database - Redis;
2) Python - Redis, PublicSuffixList, trie...

To do a prerun, type ./run.sh -p 
To do a load, type ./run.sh -l 
To do a run, type ./run.sh -r 
To do a clean up, type ./run.sh -c
